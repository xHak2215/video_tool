# video tool

### зависимости
- `python3`
- `ffmpeg`
- `moviepy` (_python библиотека_)

### функционал:
- конвертация форматов 
- сжатие видео
- удаление звука из видео
- чтение и редактирование мета данных

### команды и использование
**структура:**
[`BIN`] [`FAILE`] [`COMMAND`] [`OPTIONS`] [`ARG`]

`BIN` - бинарный файл программы или запущеный с помощью `python3 main.py` файл исходник
`FAILE` - изначальный файл с которым ведется работа 
`COMMAND` - команда(действие) которая будет определять как программа будет реагировать на файл
`OPTIONS` - дополнительные данные "настройки" работы команды
`ARG` - глобальный аргумент работающий на все команды

**команды:**
 - `info` - информация об видео такая как вес файла, расширение, FPS, кодек.

 - `optimization` (сокр. `opt`) - сжатие видео путем уменьшения разрешения и FPS. 1 аргумент после команды это % сжатия (целое число).

 - `without_audio` - удаление аудио из видео.

 - `metadata` - работа с мета данными. (без аргументов для чтения)
        аргументы:
          1 аргумент после команды это изменяемый парамитер (title, artist, date и т.д) с вносимыми данными. пример: `venv/bin/python3 main.py test.mp4 metadata title=test_data`

 - `to` - преобразование формата и переименование(опционально). 1 аргумент это имя файла с преобразованым форматом.

 - `extrude_audio` - извлечение аудио. 1 аргумент название сохраняемого аудио файла.

 **аргументы:**
 - `file_name` - имя нового сохраняемого файла, применение: `file_name=new_video.mp4` по умолчанию `output.mp4`.
